= 將發生故障的啟動介質元件傳回NetApp - FAS70 和 FAS90
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
如果您的 FAS70 或 FAS90 儲存系統中的某個元件發生故障，請將故障元件退回NetApp。請參閱 https://mysupport.netapp.com/site/info/rma["零件退貨與更換"]頁面以獲取更多資訊。
