= 自動開機媒體恢復概述： ASA A70 和 ASA A90
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以透過自動開機媒體恢復（ BMR ）選項來更換故障的開機媒體。

自動開機媒體還原會使用來自合作夥伴節點的開機映像、並自動執行適當的開機功能表選項、以便在更換的開機媒體上安裝開機映像。
