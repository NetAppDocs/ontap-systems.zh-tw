= 將故障零件退回 NetApp - ASA A70 和 ASA A90
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
如果您的ASA 70-90 系統中的某個元件發生故障，請將故障元件退回給NetApp。請參閱 https://mysupport.netapp.com/site/info/rma["零件退貨與更換"]頁面以獲取更多資訊。
