= 將故障零件歸還給NetApp AFF - E4A220和FAS2700
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
如套件隨附的RMA指示所述、將故障零件退回NetApp。如 https://mysupport.netapp.com/site/info/rma["零件退貨與更換"]需詳細資訊、請參閱頁面。

FAS2700 系統僅支援手動啟動媒體復原程序。不支援自動啟動媒體恢復。
