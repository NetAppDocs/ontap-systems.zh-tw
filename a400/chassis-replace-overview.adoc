= 機箱更換總覽- AFF VA400
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
若要更換機箱、您必須將風扇和控制器模組從受損的機箱移至與受損機箱相同機型的新機箱。

系統中的所有其他元件都必須正常運作；否則、您必須聯絡技術支援部門。

* 您可以將此程序用於ONTAP 系統支援的所有版本的功能。
* 此程序會造成中斷。對於雙節點叢集、您將會在多節點叢集中發生完整的服務中斷和部分中斷。
