= 將故障零件歸還給NetApp AFF - NetApp -即可
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
更換啟動媒體後，將故障零件退回NetApp 。AFF A300系統僅支援手動啟動媒體復原程式。不支援自動啟動媒體恢復。

如套件隨附的RMA指示所述、將故障零件退回NetApp。如 https://mysupport.netapp.com/site/info/rma["零件退貨與更換"]需詳細資訊、請參閱頁面。
