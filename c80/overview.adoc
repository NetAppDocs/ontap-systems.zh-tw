= AFF C80 的主要規格
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
以下是單一高可用性對中AFF C80 儲存系統的精選規格。請造訪NetApp Hardware Universe (HWU) 以了解此儲存系統的完整規格。



== AFF C80 的主要規格

* 平台配置： AFF C80 單機箱 HA 對
* 最大原容量：14.7360 PB
* 記憶體：512.0000 GB
* 外形尺寸：4U 機箱，配備 2 個 HA 控制器和 48 個驅動器插槽
* ONTAP版本： ONTAP：9.16.1P2
* PCIe 擴充槽：18
* 最低ONTAP版本： ONTAP 9.16.1RC1 == 橫向擴展最大值
* 類型：NAS；HA 對：12；原始容量：176.8 PB / 157.0 PiB；最大記憶體：6144 GB
* 類型：SAN；HA 對：6；原始容量：88.4 PB / 78.5 PiB；最大記憶體：3072 GB
* 類型：HA 對；原始容量：14.7 PB / 13.1 PiB；最大記憶體：512.0000

* 安全/EMC/EMI認證：EAC、UKRSEPRO
* 認證安全/EMC/EMI/RoHS：BSMI、CE DoC、UKCA DoC
* 標準 EMC/EMI：BS-EN-55032、BS-EN55035、CISPR 32、EN55022、EN55024、EN55032、EN55035、EN61000-3-2、EN61000-3-3、KS 第 15 部分 A 3830 33、230 3、KS 930 類
