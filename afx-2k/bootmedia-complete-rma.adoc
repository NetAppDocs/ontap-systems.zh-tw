= 將故障零件退回 NetApp - AFX 2K
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
如果您的 AFX 2K 儲存系統中的某個元件發生故障，請將故障元件退回 NetApp。請參閱 https://mysupport.netapp.com/site/info/rma["零件退貨與更換"]頁面以取得更多資訊。
