= 更換控制器的要求 - ASA C30
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
在更換 ASA C30 儲存系統中的控制器之前，請確保滿足成功更換的必要要求。這包括驗證系統中的所有其他元件是否正常運作，確認您擁有正確的更換控制器，以及將控制器的主控台輸出儲存至文字記錄檔。

您必須檢閱控制器更換程序的要求與考量事項。

[NOTE]
====
NetApp 強烈建議您在使用的裝置上啟用主控台記錄功能，並在執行此程序時採取下列行動：

* 在維護期間保持 AutoSupport 啟用狀態。這樣一來，如果發生問題，NetApp 支援人員就能收到診斷資訊。
* 啟用所有 CLI 工作階段的工作階段記錄。這會建立維護活動期間所有命令和系統輸出的完整記錄。如需如何啟用工作階段記錄的指引，請參閱 https://kb.netapp.com/on-prem/ontap/Ontap_OS/OS-KBs/How_to_configure_PuTTY_for_optimal_connectivity_to_ONTAP_systems["如何設定 PuTTY 以獲得與 ONTAP 系統的最佳連線"^]。


====


== 需求

* 所有機櫃都必須正常運作。
* 健全的控制器必須能夠接管要更換的控制器（在本程序中稱為「受損的控制器」）。
* 您必須以相同機型類型的控制器來更換控制器。您無法只更換控制器來升級系統。
* 在此程序中，您無法變更任何磁碟機或磁碟櫃。




== 考量

* 請務必將本程序中的命令套用至正確的控制器：
+
** _減損_控制器是要更換的控制器。
** _replacement控制器是取代受損控制器的新控制器。
** _Healthy控制器是正常運作的控制器。
