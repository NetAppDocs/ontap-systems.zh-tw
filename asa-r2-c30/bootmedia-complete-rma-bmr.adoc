= 將故障零件退回 NetApp - ASA C30
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
當 ASA C30 儲存系統中的某個元件發生故障時，請將故障元件退回給 NetApp。如 https://mysupport.netapp.com/site/info/rma["零件退貨與更換"]需詳細資訊、請參閱頁面。
