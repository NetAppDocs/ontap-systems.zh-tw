= 更換風扇AFF - S4A700s
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
若要更換風扇、請移除故障的風扇模組、然後以新的風扇模組進行更換。

. 如果您尚未接地、請正確接地。
. 請查看主控台錯誤訊息或找出主機板上風扇模組的LED燈號、以識別您必須更換的風扇模組。
. 將風扇模組側邊的鎖定彈片夾住、然後將風扇模組從控制器模組中直接提出、以移除風扇模組。
