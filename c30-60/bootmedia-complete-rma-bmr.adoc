= 將失敗的啟動媒體部分回傳給NetApp - AFF C30 和AFF C60
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
如果您的AFF C30 或AFF C60 儲存系統中的某個元件發生故障，請將故障元件退回給NetApp。請參閱 https://mysupport.netapp.com/site/info/rma["零件退貨與更換"]頁面以獲取更多資訊。
