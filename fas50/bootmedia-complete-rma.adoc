= 將故障零件退回 NetApp - FAS50
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
如果 FAS50 系統中的元件故障，請將故障零件退回 NetApp 。如 https://mysupport.netapp.com/site/info/rma["零件退貨與更換"]需詳細資訊、請參閱頁面。
