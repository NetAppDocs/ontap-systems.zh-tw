= 將故障零件退回 NetApp - ASA A20 ， ASA A30 和 ASA A50
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
當 ASA A20 ， ASA A30 或 ASA A50 儲存系統中的元件故障時，請將故障零件退回 NetApp 。如 https://mysupport.netapp.com/site/info/rma["零件退貨與更換"]需詳細資訊、請參閱頁面。
