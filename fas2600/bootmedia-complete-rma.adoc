= 將故障部件退回NetApp - FAS2600
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
請依照套件隨附的 RMA 說明中的說明，將故障元件退回給NetApp。參見 https://mysupport.netapp.com/site/info/rma["零件退貨與更換"] 更多資訊請參閱頁面。FAS2600 系統僅支援手動啟動媒體復原程序。不支援自動啟動媒體恢復。
