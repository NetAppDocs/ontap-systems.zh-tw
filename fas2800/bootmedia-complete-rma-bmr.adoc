= 將發生故障的啟動介質回傳給NetApp - FAS2800
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
如果您的 FAS2800 系統中的某個元件發生故障，請將故障元件退回給NetApp。請參閱 https://mysupport.netapp.com/site/info/rma["零件退貨與更換"]頁面以獲取更多資訊。
