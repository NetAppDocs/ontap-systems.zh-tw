== 安裝所需的設備

若要安裝儲存系統、您需要下列設備和工具。

* 存取網頁瀏覽器以設定儲存系統
* 靜電釋放（ ESD ）固定帶
* 手電筒
* 具備 USB/ 序列連線的筆記型電腦或主控台
* 2號十字螺絲起子




== 起重預防措施

儲存系統和儲存櫃很重。抬起和移動這些項目時請務必謹慎。
