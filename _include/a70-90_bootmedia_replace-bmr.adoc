更換啟動介質，該介質位於控制器背面的系統管理模組內。

.開始之前
* 您必須準備備用啟動介質。
* 為系統管理模組配備防靜電墊。


.步驟
. 請確認NVRAM卸載已完成再繼續。當 NV 模組上的 LED 熄滅時， NVRAM將被停用。
+
如果 LED 指示燈閃爍，請等待閃爍停止。如果閃爍持續超過 5 分鐘，請聯絡NetApp支援尋求協助。

+
image::../media/drw_a1K-70-90_nvram-led_ieops-1463.svg[NVRAM 注意與狀態 LED 位置圖]

+
[cols="1,4"]
|===


 a| 
image:../media/icon_round_1.png["編號 1"]
 a| 
NVRAM 狀態 LED



 a| 
image:../media/icon_round_2.png["編號 2"]
 a| 
NVRAM 注意 LED

|===
. 如果尚未接地，請到機殼後方正確接地。
. 斷開控制器電源：
+
** 對於交流電源，請將電源線從電源上拔下。
** 對於直流電源，請將電源模組從電源上斷開。


. 移除系統管理模組：
+
.. 移除連接到系統管理模組的所有電纜。在電纜上貼上標籤，以便在重新安裝時識別其正確的連接埠。
.. 按下線材管理臂兩側的按鈕，將線材管理臂向下旋轉。
.. 按下系統管理 CAM 按鈕。
+
凸輪桿遠離底盤。

.. 將凸輪桿完全向下旋轉，然後從控制器中取出系統管理模組。
.. 將系統管理模組放置在防靜電墊上，並確保啟動介質可觸及。


. 從系統管理模組中移除啟動媒體：
+
image::../media/drw_a70-90_boot_media_remove_replace_ieops-1367.svg[開機媒體取代圖形]

+
[cols="1,4"]
|===


 a| 
image::../media/icon_round_1.png[編號 1]
 a| 
系統管理模組 CAM 栓鎖



 a| 
image::../media/icon_round_2.png[編號 2]
 a| 
開機媒體鎖定按鈕



 a| 
image::../media/icon_round_3.png[編號 3]
 a| 
開機媒體

|===
+
.. 按下藍色鎖定按鈕。
.. 將啟動介質向上旋轉，從插槽中滑出，然後放在一邊。


. 將替換開機媒體安裝至系統管理模組：
+
.. 將開機媒體的邊緣對齊插槽外殼、然後將其輕推入插槽。
.. 朝鎖定按鈕方向向下旋轉開機媒體。
.. 按下鎖定按鈕，將開機媒體完全向下旋轉，然後放開鎖定按鈕。


. 重新安裝系統管理模組：
+
.. 將系統管理模組的邊緣與機箱開口對齊。
.. 輕輕地將模組滑入機箱，直到凸輪閂鎖開始嚙合。
.. 將凸輪鎖扣完全向上旋轉，以將模組鎖定到位。
.. 使用拆卸時建立的標籤，將電纜重新連接到系統管理模組。
.. 將纜線管理 ARM 向上旋轉至關閉位置。


. 重新連接控制器電源：
+
** 對於交流電源，請將電源線插入電源。
** 對於直流電源，請將電源模組重新連接到電源。
+
控制器在恢復供電後會自動重新啟動。



. 按下按鈕中斷啟動程序。 `Ctrl-C`停在 LOADER 提示符號處。
