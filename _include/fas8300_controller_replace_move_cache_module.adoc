更換控制器模組時、您必須將快取模組從受損的控制器模組移至替換控制器模組。


NOTE: Ver2 控制器模組在 FAS8000 中只有一個快取模組插槽。FAS8700 沒有 Ver2 控制器模組。快取模組功能不受插槽移除影響。

您可以使用下列動畫、圖例或書面步驟、將快取模組移至新的控制器模組。

.動畫-移動快取模組
video::d6a43902-0e78-40c3-a2bd-aad9012f5b94[panopto]
image::../media/drw_8300_Replace-flashcache.png[移除並更換快取模組]

.步驟
. 如果您尚未接地、請正確接地。
. 將快取模組從受損的控制器模組移至更換的控制器模組：
+
.. 按下快取模組末端的藍色釋放卡舌、將模組向上旋轉、然後從插槽中取出模組。
.. 將快取模組移至更換控制器模組上的相同插槽。
.. 將快取模組的邊緣與插槽對齊、然後將模組稍微插入插槽。
.. 將快取模組朝主機板方向向下旋轉。
.. 將手指放在快取模組的末端、用藍色按鈕向下推快取模組末端、然後提起鎖定按鈕、將快取模組鎖定到位。
