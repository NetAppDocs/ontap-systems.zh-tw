當您為 A700 HA 配對熱新增第二個機櫃時、您需要在每個控制器的插槽 3 和插槽 7 中、跨兩組支援 RoCE 的連接埠重新連接第一個機櫃。

此程序假設現有機櫃透過纜線連接到每個控制器插槽 3 中支援 RoCE 的 I/O 模組。

.步驟
. 將第一個機櫃重新佈線至每個控制器插槽 3 和插槽 7 中的兩組支援 RoCE 的連接埠。
+
.. 在控制器A上、將纜線從插槽3連接埠b（e3b）移至插槽7連接埠b（e7b）。
.. 在控制器B上重複相同的纜線移動


. 使用驗證已重新連接的機架是否已正確連接 https://mysupport.netapp.com/site/tools/tool-eula/activeiq-configadvisor["Active IQ Config Advisor"^]。
+
如果產生任何纜線錯誤、請遵循所提供的修正行動。
