= 將 NS224 機架熱新增至您的 ASA 系統
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
請依照以下工作流程步驟，將 NS224 機架熱新增至您的 ASA 系統。

.開始之前
* 此程序僅適用於直接連接的儲存設備。若要檢視交換器連接儲存設備的說明，請檢視link:cable-as-switch-attached.html["交換器連接纜線指南"]。
* 要熱新增 NS224 機櫃，您的 HA 配對必須符合特定需求。請檢閱link:hot-add-asa-requirements.html["熱新增需求和最佳實務做法"]。


.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-1.png["一"]link:hot-add-asa-requirements.html["檢閱熱新增需求"]
[role="quick-margin-para"]
確認您的 ASA 系統符合熱新增 NS224 機櫃的要求，包括 ONTAP 版本支援、機櫃數量和佈線要求。

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-2.png["二"]link:hot-add-asa-prepare.html["準備熱新增您的機櫃"]
[role="quick-margin-para"]
視您的平台機型而定、您可能需要安裝其他具備 ROCE 功能的 PCIe 卡或 I/O 模組、設定支援 ROCE 功能的非專用乙太網路連接埠以供儲存使用、在不同插槽的兩組連接埠之間重新配置現有機櫃、以因應插槽故障的恢復能力、 如果您是手動指派磁碟機擁有權、請停用自動磁碟機指派。

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-3.png["三"]link:hot-add-asa-install.html["安裝您的機櫃"]
[role="quick-margin-para"]
安裝機櫃的導軌套件、在電信機架或機櫃中安裝並固定機櫃、連接電源線以開啟機櫃電源，並指派唯一的機櫃 ID 以確保機櫃在 HA 配對中是獨特的。

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-4.png["四"]link:hot-add-asa-cable-overview.html["為您的機櫃佈線"]
[role="quick-margin-para"]
將您要熱新增的機櫃連接至 HA 配對中的每個控制器、使其有兩個連線。

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-5.png["五"]link:hot-add-asa-complete.html["完成熱新增"]
[role="quick-margin-para"]
如果您在準備熱新增時停用自動磁碟機指派、則需要手動指派磁碟機擁有權、然後視需要重新啟用自動磁碟機指派。
