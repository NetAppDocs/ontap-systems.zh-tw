= 將 NS224 機架熱新增至 AFF 系統的準備工作概述
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
準備將 NS224 機櫃熱新增至 AFF 系統，請確保有足夠的支援 RoCE 的儲存連接埠、將其設定為儲存用途，並針對新機櫃最佳化現有組態。



== 需要準備什麼

將 NS224 機架熱新增到 AFF 系統之前，需要先驗證並設定儲存連線基礎架構，然後再實際安裝新機架。這些工作可確保您的 HA 配對具有必要的支援 RoCE 的乙太網路連接埠，並且您的系統已設定為支援額外的儲存容量。

您需要完成的特定準備工作取決於您的平台型號、目前組態以及您要新增的磁碟櫃數量：

安裝具備 ROCE 功能的 PCIe 卡或 I/O 模組:: 請確認您的系統擁有足夠的支援 RoCE 的乙太網路連接埠（每個控制器的每個機櫃兩個）。如果需要更多連接埠，請在正確的控制器插槽中安裝支援 RoCE 的 PCIe 卡或 I/O 模組。
設定具備 ROCE 功能的連接埠以供儲存使用:: 如果您使用非專用的 RoCE 連接埠連接 NS224 儲存架，請將這些連接埠配置為儲存用途，而不是網路用途。此過程不會中斷服務。
可重新配置現有的機櫃:: 對於某些平台型號，在安裝額外的支援 RoCE 的 PCIe 卡或 I/O 模組後，您可能需要將現有機櫃重新連接至不同插槽中的兩組連接埠，以提供防止插槽故障的備援能力。
停用自動磁碟機指派:: 如果您打算手動指派磁碟機所有權，請在熱新增磁碟櫃之前停用自動磁碟機指派。
