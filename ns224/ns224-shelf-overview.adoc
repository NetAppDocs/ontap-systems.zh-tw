= 概覽 - NS224 機櫃
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
NetApp NS224 磁碟機櫃是 2U 機架單元、最多可容納 24 個磁碟機。NS224 磁碟機櫃與 2.5 吋小型磁碟機相容、同時支援 SAS 和 SATA 介面。

請採取下列行動來維護 NS224 磁碟機櫃。

* link:hot-remove-shelf.html["熱移除機櫃"]
