= 開機媒體恢復總覽 - ASA A1K
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
ASA A1K 儲存系統的開機媒體恢復會使用合作夥伴節點的開機映像，並自動執行適當的開機功能表選項，在替換的開機媒體上安裝開機映像。

當您遇到開機錯誤訊息時，需要更換開機媒體，並從合作夥伴節點還原 ONTAP 映像。
