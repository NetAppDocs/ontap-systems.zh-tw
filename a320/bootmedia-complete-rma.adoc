= 將故障零件歸還給NetApp AFF - NetApp -即可
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
請依照套件隨附的 RMA 說明中的說明，將故障元件退回給NetApp。參見 https://mysupport.netapp.com/site/info/rma["零件退貨與更換"] 更多資訊請參閱頁面。AFF A320系統僅支援手動啟動媒體復原程式。
