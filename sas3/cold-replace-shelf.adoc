= 冷更換 DS212C、DS224C 或 DS460C 層架
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
在生產系統中更換正在使用的磁碟機磁碟架時，需要執行冷磁碟架更換。這是一個會中斷系統運作的操作，需要您停止 HA 配對中的控制器。

此程序適用於具有 IOM12、IOM12B 或 IOM12C 模組的磁碟櫃。

請使用 NetApp 知識庫文件 https://kb.netapp.com/onprem/ontap/hardware/How_to_replace_a_shelf_chassis_using_a_cold_shelf_removal_procedure["如何使用冷機櫃移除程序來更換機櫃機箱"]。
